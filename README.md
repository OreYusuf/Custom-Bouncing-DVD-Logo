### Prerequisites

* Install Npm
  ```sh
  npm install npm@latest -g
  ```

### Local Preview

1. Clone the repo
   ```sh
   git clone https://github.com/OreYusuf/Custom-Bouncing-DVD-Logo
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run locally
     ```sh
    npm start
   ```
